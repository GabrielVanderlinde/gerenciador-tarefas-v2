 FUNÇÃO: main()
 ┌─────────────────────────────────────────────────────┐
 │  MENU PRINCIPAL                                     │
 │  Exibição: a cada iteração do loop                  │
 └─────────────────────────────────────────────────────┘
  → Saída para o usuário:
      "Gerenciador de Tarefas"
      "1 - Adicionar tarefa"
      "2 - Listar tarefas"
      "3 - Concluir tarefa"
      "4 - Remover tarefa"
      "5 - Listar pendentes"
      "0 - Sair"
      "Opção: "

 ┌─────────────────────────────────────────────────────┐
 │  CASE 1 — Adicionar Tarefa                          │
 │  Entrada de dados solicitada ao usuário             │
 └─────────────────────────────────────────────────────┘
  → Prompts de entrada:
      "Título: "
      "Descrição: "
      "Data (formato dd/MM/yyyy): "

 ┌─────────────────────────────────────────────────────┐
 │  CASE 3 — Concluir Tarefa                           │
 │  Entrada de dados solicitada ao usuário             │
 └─────────────────────────────────────────────────────┘
  → Prompt de entrada:
      "Adicione o Id da Tarefa: "

 ┌─────────────────────────────────────────────────────┐
 │  CASE 4 — Remover Tarefa                            │
 │  Entrada de dados solicitada ao usuário             │
 └─────────────────────────────────────────────────────┘
  → Prompt de entrada:
      "Digite o ID da tarefa para remover: "

 ┌─────────────────────────────────────────────────────┐
 │  CASE 0 — Sair                                      │
 │  Encerramento do programa                           │
 └─────────────────────────────────────────────────────┘
  → Saída para o usuário:
      "Encerrando Programa..."

 ┌─────────────────────────────────────────────────────┐
 │  DEFAULT — Opção inválida                           │
 │  Quando o usuário digita uma opção fora do menu     │
 └─────────────────────────────────────────────────────┘
  → Saída para o usuário:
      "Opção Inválida!"

 FUNÇÃO: toString()
 ┌─────────────────────────────────────────────────────┐
 │  Exibição da Tarefa                                 │
 │  Chamada por listarTarefas() e listarPendentes()    │
 └─────────────────────────────────────────────────────┘
  → Status possíveis:
      "[TAREFA CONCLUÍDA]"
      "[TAREFA PENDENTE]"

  → Formato completo exibido:
      "[STATUS] ID: {id} | {titulo} | {descricao} | Data: {dd/MM/yyyy}"

  → Exemplo de saída:
      "[TAREFA PENDENTE] ID: 1 | Estudar Java | Revisar collections | Data: 30/04/2025"
      "[TAREFA CONCLUÍDA] ID: 2 | Reunião | Apresentar projeto | Data: 15/04/2025"
